# Identical to requirements.txt except we use onnxruntime-directml instead of
# the CPU-only onnxruntime package. onnxruntime-directml ships the DirectML
# execution provider, which gives GPU acceleration on any DX12-compatible GPU

# the `onnxruntime` module name — so only one may be installed at a time.
#
# Local dev installs still use requirements.txt (CPU-only) to avoid dragging
# GPU drivers into the dev loop. This file is referenced by the Windows build
# in .github/workflows/main.yml.
